本项目主要目的是利用GFW漏洞，为PC和iOS设备提供一个最为便捷(但依赖个人服务器)的一个翻墙代理方案。以黑名单的方式记录需要代理的网站列表，最小化代理服务器的压力。

使用方法

  • PAC: http://nav.liruqi.info/gfw.pac
  • iPhone/iPad 教程
  • Mac OS X 教程
  • Windows: Internet选项 -> 连接 选项卡 -> 局域网设置(如果是电脑拨号上网, 这里点'设置') -> 使用自动配置脚本 -> 填入脚本地址: http://nav.liruqi.info/gfw.pac -> 确定
  • 不建议使用全局代理，国内速度慢，流量大了可能会禁止代理国内网站
  • 国内无需密码即可访问。非大陆用户可能会提示输入用户名密码: nine/passwords

如果希望在本页面列出个人的代理服务器，请fork本项目并加入相关配置信息，再发pull request。

其它代理方案推荐

本项目局限性在于依赖系统代理。有些应用如Facebook iOS app，不走系统代理。遇到这种问题，可以选择其它更底层的网络代理:
  • VPN Gate: 公共VPN中继服务器, 支持多个VPN协议。
  • fqrouter: 目前主要在做Android 平台的系统代理以及无线中继(只有一部分Android设备支持中继模式)。
  • shadowsocks: 一个加密的 socks5 代理隧道，网站可以购买付费服务。shadowsocks.net上可以免费领取网友分享的账号。
